# Console Log Label

Babel plugin for javascript label that log change into console.log()

### Before

```javascript
log: "Hello World"
```

### After

```javascript
console.log("Hello World")
```

# Uses

```command
npm install --save-dev babel-plugin-console-log-label
```

# .babelrc

```JSON
{
  "plugins": [
    "console-log-label"
  ]
}
```
